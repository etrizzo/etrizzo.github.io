Stepping Stones Collaboration - Game Design Final Project

About Stepping Stones

Stepping Stones is a block-placing puzzle game where you must create a staircase to save your friend rapunzel-style from a high tower. However, if two blocks of the same color touch, they will both disappear.

This game was designed and developed over six weeks by Emily Rizzo, Linnea Kirby, Andres Cuervo, Alejandro Belgrave and Margaret McCarthy. This was the final project of our group for our Game Design course. Within the project, my primary responsibilities included the 'gravity' logic that causes blocks to fall, the growing blocks encountered in later levels, and developing a level reader to generate levels from text files.

Game Details

  • If two blocks of the same color touch, both will be destroyed
  • Use the queue beneath your character to see upcoming blocks
  • Your character can only climb or fall 1 level at a time
  • When you believe your path is complete, test the path (top left) to progress
  • Click on movable blocks to pick them up, and click again to place them. They can float!
  • Some blocks will grow when placed to form rigid shapes, which can be used to form bridges
  • If one square in a rigid shape breaks, the shape will fall apart!
